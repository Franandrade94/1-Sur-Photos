# 📸 Photo & Video Portfolio — Panama

Welcome to the personal portfolio of a visual artist based in Panama who specializes in photography and videography. This website was designed and developed as a fully responsive platform to showcase her work, style, and visual storytelling across various projects and formats.

---

## ✨ Features

- 🌐 Fully responsive design for mobile, tablet, and desktop
- 🎞️ Photo and video gallery, organized by type
- ⚡ Smooth animations and transitions
- 🖼️ Lazy loading for optimized performance
- 🔍 Modal preview for high-resolution viewing
- 🧭 Clean navigation with a dynamic scroll-aware navbar
- 🎯 Built with accessibility and usability in mind

---

## 🚀 Tech Stack

- **React** (Vite) for frontend development
- **React Router** for page navigation
- **React LazyLoad** for progressive image loading
- **React Modal** for interactive image previews
- **CSS** (custom) for layout and styling
- **ClipLoader (react-spinners)** for loading state
